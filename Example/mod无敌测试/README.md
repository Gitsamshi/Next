# 觅长生 - 主角无敌Mod

## 简介

这是一个让主角从游戏开始就处于无敌状态的Mod。启用后，主角将免疫所有伤害，让你可以轻松体验游戏内容。

## 功能特性

- 自动在进入游戏时为主角添加无敌Buff
- 可自定义无敌Buff的层数（1-100层）
- 可选择是否显示无敌效果启用提示
- 随时可以在Mod设置中启用或关闭无敌模式

## 安装方法

### 方法一：复制到游戏目录（推荐）

1. 将整个 `mod无敌测试` 文件夹复制到游戏目录：
   ```
   觅长生/本地Mod测试/mod无敌测试
   ```

2. 启动游戏，在Mod管理界面启用"主角无敌Mod"

3. 开始新游戏或加载存档，无敌效果将自动生效

### 方法二：使用Mod管理器

1. 如果游戏支持Mod管理器，可以直接导入此Mod文件夹

2. 在Mod管理界面启用"主角无敌Mod"

## 使用说明

### 启用无敌效果

1. 进入游戏的Mod设置界面
2. 找到"主角无敌Mod"
3. 确保"启用无敌模式"开关已打开
4. 调整"无敌Buff层数"（默认10层，可设置1-100层）
5. 进入游戏后，无敌效果将自动添加

### 关闭无敌效果

1. 在Mod设置中关闭"启用无敌模式"
2. 重新加载游戏，无敌效果将不再自动添加
3. 如果想移除当前角色的无敌效果，需要存档后重新开始

### 配置选项

- **启用无敌模式**：控制是否在进入游戏时自动添加无敌效果（默认：开启）
- **无敌Buff层数**：设置无敌Buff的叠加层数，层数越多效果越强（默认：10层）
- **显示启用提示**：控制进入游戏时是否显示无敌效果已启用的提示信息（默认：开启）

## 技术说明

### 实现原理

本Mod通过以下方式实现无敌效果：

1. **自定义Buff**：创建了一个ID为999的无敌Buff
2. **Buff效果**：Buff效果ID为9999，设置了极高的减伤值（-999999999）
3. **自动触发**：使用"进入游戏"触发器，在玩家进入游戏时自动执行
4. **条件判断**：通过设置中的"启用无敌模式"开关控制是否启用
5. **先天Buff**：使用先天Buff系统，确保Buff持续存在

### Buff数据说明

- **Buff ID**: 999
- **Buff名称**: 无敌
- **效果ID**: 9999
- **持续时间**: 999999回合（接近永久）
- **减伤值**: -999999999（负值表示减少受到的伤害）

### 自定义修改

如果你想调整无敌效果，可以修改以下文件：

1. **调整Buff持续时间**：编辑 `Data/BuffJsonData/999.json` 的 `totaltime` 字段
2. **调整减伤强度**：编辑 `Data/BuffSeidJsonData/9999.json` 的 `value1` 字段
3. **修改触发条件**：编辑 `NData/DialogTrigger/trigger.json` 的 `condition` 字段
4. **自定义提示信息**：编辑 `NData/DialogEvent/invincible.json` 的 `dialog` 数组

## 注意事项

1. 本Mod会使游戏失去挑战性，建议仅在体验剧情或测试时使用
2. 无敌效果只影响主角，不影响NPC和宠物
3. 某些特殊剧情可能会强制设定角色状态，可能导致无敌效果暂时失效
4. 建议在新游戏时启用，避免对现有存档造成影响
5. 如遇到问题，可以尝试关闭Mod后重新启动游戏

## 兼容性

- 与大部分其他Mod兼容
- 不会修改游戏原有数据，只是添加新的Buff
- 卸载后不会对游戏造成负面影响
